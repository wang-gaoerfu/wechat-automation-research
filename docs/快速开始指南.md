## 一、环境准备

### 1.1 基础要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Windows 10/11 或 Linux |
| **Python** | 3.9 或更高版本 |
| **内存** | 最少 4GB，推荐 8GB |
| **硬盘** | 最少 10GB 可用空间 |
| **网络** | 可访问互联网 |

### 1.2 安装 Python

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.9 或更高版本（推荐 3.11）
3. 运行安装程序
4. **重要**：勾选 "Add Python to PATH"
5. 点击 "Install Now"

验证安装：
```bash
python --version
pip --version
```

### 1.3 安装 Git（可选）

用于版本控制：
1. 访问 https://git-scm.com/download/win
2. 下载并安装 Git for Windows
3. 安装时保持默认选项即可

验证安装：
```bash
git --version
```

## 二、企业微信方案

### 2.1 准备企业微信账号

#### 步骤1：注册企业微信

1. 访问 https://work.weixin.qq.com/
2. 点击 "注册"
3. 选择 "企业" 或 "个人工商户"
4. 填写企业信息完成注册

#### 步骤2：创建应用

1. 登录企业微信管理后台：https://work.weixin.qq.com/wework_admin/
2. 进入 "应用管理"
3. 点击 "创建应用"
4. 填写应用信息：
   - 应用名称：如 "微信自动化助手"
   - 应用描述：自动化运营工具
   - 应用功能：勾选 "消息订阅"
5. 点击 "创建"

#### 步骤3：获取应用凭证

在应用详情页获取：
- `AgentId`（应用ID）
- `Secret`（应用密钥）

#### 步骤4：配置回调地址（接收消息用）

1. 在应用详情页，点击 "接收消息" 配置
2. 设置 "URL" 为你的服务器地址，如：`https://your-domain.com/wx/callback`
3. 设置 "Token"（自己生成随机字符串）
4. 设置 "EncodingAESKey"（点击"随机生成"）
5. 保存配置

#### 步骤5：获取企业 CorpID

1. 登录企业微信管理后台
2. 进入 "我的企业" -> "企业信息"
3. 在底部获取 `CorpID`

### 2.2 安装与配置

#### 步骤1：下载项目

```bash
cd D:\my_project\ai_projects
git clone https://github.com/wang-gaoerfu/wechat-automation-research.git
cd wechat-automation-research\enterprise-wechat
```

#### 步骤2：创建虚拟环境

```bash
python -m venv venv
```

#### 步骤3：激活虚拟环境

**Windows：**
```bash
venv\Scripts\activate
```

**Linux/Mac：**
```bash
source venv/bin/activate
```

#### 步骤4：安装依赖

```bash
pip install -r requirements.txt
```

#### 步骤5：配置

```bash
cp config\config.yaml.example config\config.yaml
```

编辑 `config\config.yaml`：

```yaml
wechat:
  corp_id: "ww你的企业ID"
  corp_secret: "你的应用Secret"
  agent_id: "你的应用AgentId"
  callback_token: "你设置的Token"
  callback_aes_key: "你设置的EncodingAESKey"

database:
  url: "sqlite:///./wechat.db"

app:
  host: "0.0.0.0"
  port: 8000
  debug: false
```

### 2.3 启动服务

```bash
python run.py
```

或使用 uvicorn：

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2.4 验证运行

打开浏览器访问：http://localhost:8000/docs

应该看到 FastAPI 的 API 文档页面。

## 三、个人微信方案

### 3.1 环境准备

#### 步骤1：安装 Python 依赖

```bash
cd D:\my_project\ai_projects\wechat-automation-research\personal-wechat
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

#### 步骤2：安装 WeChatFerry

1. 下载 WeChatFerry：
   - 访问 https://github.com/starRTC/WeChatFerry/releases
   - 下载最新版本的 Windows DLL

2. 或通过 pip 安装 Python 客户端：
```bash
pip install python-wechatferry
```

#### 步骤3：安装微信

1. 下载并安装微信 PC 版：https://pc.weixin.qq.com/
2. **支持的版本**：
   - 3.9.10.19
   - 3.9.8.15
   - 3.9.6.29
3. 登录微信并保持 PC 端在线

#### 步骤4：注入 DLL（关键步骤）

1. 打开微信 PC 版并登录
2. 运行 WeChatFerry 的注入工具
3. 等待注入成功提示
4. 确认 WeChatFerry 服务运行在 localhost:10086

### 3.2 配置

```bash
cp config\config.yaml.example config\config.yaml
```

编辑 `config\config.yaml`：

```yaml
wcf:
  host: "localhost"
  port: 10086
  reconnect_interval: 30
  max_reconnect_attempts: 5

anti_ban:
  daily_message_limit: 500
  min_send_interval: 3
  rest_hours:
    start: 2
    end: 6

database:
  url: "sqlite:///./personal_wechat.db"

app:
  host: "0.0.0.0"
  port: 8001
  debug: false
```

### 3.3 启动服务

```bash
python run.py
```

### 3.4 验证运行

访问：http://localhost:8001/docs

## 四、API 接口说明

### 4.1 企业微信 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `GET /health` | GET | 健康检查 |
| `GET /api/v1/wechat/token` | GET | 获取访问令牌 |
| `POST /api/v1/message/send` | POST | 发送文本消息 |
| `POST /api/v1/message/send_markdown` | POST | 发送 Markdown |
| `POST /api/v1/message/send_group` | POST | 群发消息 |
| `POST /wx/callback` | POST | 接收微信回调 |
| `GET /api/v1/customer/list` | GET | 获取客户列表 |
| `GET /api/v1/customer/{id}` | GET | 获取客户详情 |

#### 发送消息示例

```bash
curl -X POST http://localhost:8000/api/v1/message/send \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "wangjiacheng",
    "content": "你好，这是一条测试消息"
  }'
```

## 六、常见问题

### Q1：企业微信回调配置失败？

**问题**：配置回调时验证失败

**解决**：
1. 确保服务器公网可访问
2. 使用 ngrok 本地测试：
   ```bash
   ngrok http 8000
   ```
3. 用 ngrok 提供的 URL 替换回调地址

### Q2：WeChatFerry 连接失败？

**问题**：无法连接到 localhost:10086

**解决**：
1. 确认 WeChatFerry 服务已启动
2. 确认微信已登录并注入 DLL
3. 检查端口是否被占用：`netstat -an | findstr 10086`

### Q3：消息发送失败？

**问题**：返回错误码

**解决**：
1. 检查 access_token 是否有效
2. 确认目标用户是否存在
3. 检查发送频率是否超限

### Q4：个人微信被封号？

**问题**：发送消息后账号异常

**解决**：
1. 降低发送频率
2. 增加消息内容多样性
3. 避免夜间发送
4. 使用企业微信方案（无封号风险）

## 七、安全建议

1. **不要泄露** CORP_ID、CORP_SECRET
2. 生产环境使用环境变量而非配置文件
3. 启用 HTTPS
4. 定期更新依赖包
5. 企业微信方案作为首选方案
