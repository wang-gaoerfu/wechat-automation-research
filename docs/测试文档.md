# 微信自动化运营系统测试文档

**方案设计者：墨飞科技**
**测试日期：2026年5月25日**

---

## 一、测试概述

### 1.1 文档目的

本文档定义微信自动化运营系统的测试用例，包含测试条件、输入数据、预期输出，用于指导测试执行和验收。

### 1.2 测试范围

| 系统 | 测试范围 |
|------|---------|
| **企业微信方案** | Token管理、消息发送、回调处理、客户管理 |
| **个人微信方案** | WCF连接、防风控（频率限制、内容生成、行为模拟）、消息处理 |

### 1.3 测试环境要求

**企业微信方案：**
- Python 3.9+
- 企业微信已开通（需 CORP_ID、CORP_SECRET）
- 可访问 `https://qyapi.weixin.qq.com`

**个人微信方案：**
- Python 3.9+
- Windows PC 已安装微信 3.9.10.19 / 3.9.8.15 / 3.9.6.29
- WeChatFerry DLL 已注入
